## Project Overview

This is a static personal portfolio website for Yuqi Sun (Kevin), hosted on GitHub Pages. There is no build step — all HTML, CSS, and JS files are served directly as-is.

## Development

**Serving locally:**
```bash
python3 -m http.server 8000
# Then open http://localhost:8000
```

**Deploying:** Push to `main` branch. GitHub Pages automatically serves the site.

## Architecture

### Pages
All top-level `.html` files are standalone pages linked via a shared navigation header:
- `index.html` — Home/About
- `resume.html` — Resume (also links to `resume.pdf`)
- `blog.html` — Blog posts
- `projects.html` — Project showcase
- `showcase.html` — Interactive games index
- `asteroid.html` — Vanilla JS asteroids game
- `invaders.html` / `civilisation.html` — Wrappers for Construct 3 game exports

### Styling
`styles.css` is the single shared stylesheet for all pages. It uses University of Michigan brand colors:
- Blue: `#102F58`
- Maize: `#F7D147`

Responsive breakpoints are at 600px and 700px. Max content width is 1000px.

### Games
`invaders-game/` and `civilisation_game/` are Construct 3 exported web apps. They are self-contained with their own `index.html`, `style.css`, `sw.js` (service worker), `appmanifest.json`, and compiled scripts in `scripts/`. **Do not manually edit files inside these directories** — they are generated by Construct 3.

`asteroid.html` contains a vanilla JS asteroids clone directly inline.

### Assets
`assets/` holds images, PDFs (poetry anthologies, project presentations), and blog thumbnails. Profile photos and resume PDFs live in the root directory.
